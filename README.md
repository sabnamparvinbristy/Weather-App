## ✨ Features

- 🔍 **City Search** — Search any city worldwide with instant results
- 🌡️ **Real-time Weather** — Live temperature, humidity, wind, pressure & visibility
- 📅 **5-Day Forecast** — Daily weather outlook with icons and conditions
- 🌗 **Dark / Light Mode** — Toggleable theme with smooth transitions
- 🎨 **Glassmorphism UI** — Dynamic gradients that change based on weather conditions
- 🌡️ **Unit Toggle** — Switch between °C (metric) and °F (imperial)
- 🕐 **Live Clock** — Real-time clock and date in the header
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- 🌅 **Sunrise & Sunset** — Local times calculated from timezone offset
- 💨 **Wind Direction** — Compass degree included in stats

## 🎨 Weather Themes

The UI dynamically changes its gradient and accent colors based on current conditions:

| Condition | Theme |
|---|---|
| ☀️ Clear | Golden amber gradients |
| ☁️ Cloudy | Cool slate blues |
| 🌧️ Rain | Deep ocean blues |
| ⛈️ Thunderstorm | Dark violet purples |
| ❄️ Snow | Icy sky blues |
| 🌫️ Mist | Soft grey-blue haze |
